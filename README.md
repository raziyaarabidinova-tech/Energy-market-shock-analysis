# Energy Market Price Analysis

## Problem
This project analyzes recent energy market price dynamics and how prices evolve over time.

## Data
Daily energy price indices for different markets, normalized for comparison.

## Method
- Price normalization (Base = 100)
- Time series analysis
- Comparison across markets

## Key Insights
- Markets show strong price movements over time
- Differences in dynamics across regions
- Increasing divergence after recent market movements

## Tools
Python, Pandas, Matplotlib
